Conditional Control Statements:-

If Statement

The statement will go inside the block when the condition is true.otherwise it will not execute this block.This is also called a control statement which is use to control the execution of flow.
Syntax of if statement:
if(condition){
// statements (if Block)
} //other statements

If-else Statement

This statement has two block,when the condition is true the statement will go inside the if statement otherwise they will execute the else statement.
Syntax of if-else statement:

if(condition){
// statements (if Block) }
else{
// statements (Else block) }
//other statements

If-else-if Statement

In this statement if your condition is true then it will go inside the if block.otherwise,They will check else if statement and if the condition is still not true then they will execute the else block.

Syntax of if-else if statement :

if(condition){
// statements (if Block) }
else if{
// statements (Else-If block) }
else{
//statements(Else Block) }
//other statements
